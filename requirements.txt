#   - psycopg2-binary works on Apple Silicon
#   - torch: Use conda for better performance

#   - All packages work out of the box
#   - For production, compile psycopg2 from source

# ============================================================================
# Troubleshooting
# ============================================================================
#
# Issue: psycopg2 installation fails
# Solution: Use psycopg2-binary OR install PostgreSQL dev headers
#   Ubuntu/Debian: sudo apt-get install libpq-dev
#   CentOS/RHEL: sudo yum install postgresql-devel
#   macOS: brew install postgresql
#
# Issue: torch too large
# Solution: Install CPU-only version
#   pip install torch --index-url https://download.pytorch.org/whl/cpu
#
# Issue: ImportError for a package
# Solution: Reinstall in a fresh virtual environment
#   python -m venv venv
#   source venv/bin/activate  # On Windows: venv\Scripts\activate
#   pip install --upgrade pip
#   pip install -r requirements.txt
#
